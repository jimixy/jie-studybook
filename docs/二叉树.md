# 二叉树

和链表一样，属于动态数据结构

## 特点

- 二叉树具有唯一的根节点
- 二叉树每个节点最多有两个孩子
- 二叉树具有天然递归结构

## 二叉树的遍历

- 深度遍历(DFS)  
  按是否递归来分：

  - 递归
  - 非递归

  按遍历顺序来分：

  - 前序遍历(根节点->左子树->右子树)
  - 中序遍历(左子树->根节点->右子树)
  - 后序遍历(左子树->右子树->根节点)

- 广度遍历(BFS)  
  按是否递归来分：
  - 递归
  - 非递归

## 知识点

1. 搜索二叉树的中序遍历结果是有序的
2. BFS 找到的路径是最短的
